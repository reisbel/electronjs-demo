# electronjs-demo

Demo application built with [Electronjs](http://electronjs.org).

## Build steps

Install nodejs

Clone repository

```bash
git clone https://github.com/reisbel/electronjs-demo.git && cd electronjs-demo
```

Install packages

```bash
npm install
```

Run the application

```bash
npm start
```
